# AutoNetworkConnect (自动断网重连工具)

AutoNetworkConnect 是一款专为需要网页认证的网络环境（如典型的校园网、企业内网、公共 Wi-Fi 等 Captive Portal 环境）打造的自动连网工具。

当你身处这些网络环境时，通常需要手动打开网页并输入账号密码才能上网。如果网络不稳定掉线，或者长时间无操作被踢下线，频繁地手动重新登录会非常繁琐。**AutoNetworkConnect** 能够定时在后台检测你的网络状态，并定时、自动向认证服务器发送登录请求，让你始终保持网络畅通。

本项目已打包，包含了所有必要的运行库（dll），**无需任何环境配置，直接双击运行 exe 文件即可使用**。


## 🚀 使用教程

使用本工具非常简单，你只需要通过浏览器抓取一次你登录网络时的请求（cURL），剩下的工作都可以交给软件自动完成。

### 第一步：获取登录的 cURL 命令
1. 连接上你的校园网（或需要认证的网络），来到网络登录页面。
2. 在浏览器中按下 `F12` 键，打开**开发者工具**。
3. 切换到 **“网络” (Network)** 标签页，或一个无线网图标的页面。为了避免干扰，你可以点击一下清除按钮（🚫图标）清空现有记录。
4. 在网页中正常输入你的账号和密码，然后点击网页上的“登录”按钮。
5. 在开发者工具的网络列表中，找到刚才发送的登录请求记录（通常是最上面的那条，或者名称类似 `login`、`auth` 的 POST 请求）。
6. **右键**点击该记录 -> 选择 **“复制” (Copy)** -> 点击 **“复制为 cURL (bash)” (Copy as cURL)**。

### 第二步：在软件中配置并运行
1. 打开 `AutoNetworkConnect.exe`。
2. 将刚才复制的 cURL 命令粘贴到软件下方的 **“curl抓包输入”** 文本框中。
3. 点击 **“解析Curl”** 按钮。此时软件会自动从你的 cURL 中提取出账号和密码，并自动填入上方的“账号”和“密码”输入框中。
   > **提示**：如果你想使用其他账号登录，可以直接修改上方文本框中的“账号”和“密码”。当你点击“开始”时，软件会自动用输入框里的内容**覆盖** cURL 中对应的字段信息。
4. 设置 **“轮询间隔”**：这决定了软件每隔多长时间检测一次网络状态（例如 0:00:30 表示每 30 秒检测一次，建议不要设置得太短以免给服务器造成压力）。
5. 点击 **“开始”**。软件主界面会自动隐藏，转入后台运行并弹出日志窗口，持续监控你的网络。

## ⚙️ 选项说明

- **☑️ 开机自启**：勾选后，软件将在 Windows 启动时自动运行，并且会直接读取你上一次的配置**自动开始**网络监控。配合隐藏在后台的特性，实现开机自动连网。
- **☑️ 详细日志**：勾选后，点击开始弹出的日志窗口中会显示更详细的运行状态（包括每次 Ping 服务器返回的详细代码等）。如果遇到配置了依然连不上网的情况，可以勾选此项来排查问题。
